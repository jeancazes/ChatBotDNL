# Jeu avec Chatbot DNL

Application pedagogique de jeu de role avec chatbot pour les sections SELO/DNL (Anglais, Espagnol, Allemand).

## Lancement en local

```bash
npm install
npm run dev
```
Ouvrez http://localhost:5173

## Deploiement (Vercel)

Connectez ce repo a Vercel — le build est configure automatiquement.

## Configuration

Au premier lancement, cliquez **Cles API** et saisissez :
- Cle Anthropic (obligatoire) : https://console.anthropic.com/settings/keys
- Cle OpenAI (mode oral uniquement) : https://platform.openai.com/api-keys

## Scenarios inclus

1. Le vaccinosceptique
2. Le climatosceptique
3. Le debat OGM
4. La planete dans notre assiette
